# 概要
- Dense Deck Poker で利用しているハンド配布アルゴリズムについてのまとめ
- 重み付け抽選と棄却法の組み合わせによって、単純なランダム配布ではなく、「一定以上の強さを持つハンドが配布されやすいが、ゲーム全体としての公平性・確率分布は通常NLHMと同じ状態」の実現を目指す

# アルゴリズムの目的
## 通常のポーカー
- ハンド配布・ボードカードは、52枚のデックから完全ランダムに決定される
- 結果として、「ゲームに参加すると期待値がマイナスになる」という弱ハンド群が大量にプレイヤーに配布される
- ゲームに参加しない、思考しない時間が増えてしまう

## Dense Deck Poker
- 通常のポーカーよりも、思考の密度を上昇させたい
- ゲームに参加可能な強さを持つハンドの配布確率が上がり、プレイヤーの思考密度の上昇を意図する
- Short Deck （5以下のランクをデックから排除する）などの方法では、ハンドの相対的な強さは変わらないため、不参加ハンド群は一定存在し続けてしまう

# アルゴリズムの全体構造
- 全体は以下の2段階
```
① 候補ハンド生成（重み付き）
② 整合性チェック（棄却法）
```
