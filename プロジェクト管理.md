# 概要
```
project
┣ /infra # AWS CDK
└ /src # tsファイル
```
- この場合、 `infra` と `src` は別プロジェクトで依存関係を管理する

## `src` 用のプロジェクト
- ルートディレクトリで以下コマンドを実行する
```bash
pnpm init
pnpm add fastify
pnpm add -D typescript ts-node @types/node
npx tsc --init
```
- `package.json` に以下を追記
```json
{
  "scripts": {
    "dev": "ts-node src/app.ts"
  }
}
```
